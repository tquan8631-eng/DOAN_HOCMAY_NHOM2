DOAN_HOCMAY_NHOM2

Giới thiệu

Dự án học máy sử dụng các thuật toán phân cụm và phân tích dữ liệu.

Yêu cầu hệ thống

Python 3.8 trở lên

pip (Python package manager)

Hướng dẫn cài đặt
1. Clone repository
   
git clone https://github.com/tquan8631-eng/DOAN_HOCMAY_NHOM2.git

cd DOAN_HOCMAY_NHOM2

2. Tạo môi trường ảo (Virtual Environment)
   
Trên Windows:

python -m venv venv

venv\Scripts\activate

Trên macOS/Linux:

python -m venv venv

source venv/bin/activate

3. Cài đặt các thư viện cần thiết
   
pip install -r requirements.txt

Nếu chưa có file requirements.txt, cài đặt các thư viện cơ bản:

pip install numpy pandas scikit-learn matplotlib seaborn



Cách chạy chương trình

Chạy file chính

cd src

python main.py
